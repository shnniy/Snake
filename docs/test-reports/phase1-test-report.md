## 5. 已知问题

### 5.1 已修复问题
1. **RejectedExecutionException**：当 pause() 与 tick() 线程存在竞态时，`scheduleTick()` 中的 `executor.schedule()` 可能抛出异常。已通过 try-catch 处理。
2. **方向缓冲区测试**：测试中使用了与初始方向相反的方向，导致方向被拒绝。已修正测试逻辑。

### 5.2 当前无未解决问题
